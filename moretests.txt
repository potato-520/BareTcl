# Test Group B

## Massive Variable Churn

目标：

验证变量表与GC协同。

```tcl

    set v$i $i
}
```

随后：

```tcl
for {set i 0} {$i < 50000} {incr i} {
    unset v$i
}
```

重复多轮。

要求：

* Arena不泄漏
* GC后Arena恢复
* 无悬挂引用

输出：

Peak Usage
Final Usage

# Test Group C

## Deep Proc Recursion Stress

目标：

验证FSM Frame系统。

测试：

```tcl

    if {$n < 2} {
        return $n
    }

    expr {
        [fib [expr {$n-1}]]
        +
        [fib [expr {$n-2}]]
    }
}
```

执行：

```tcl

fib 20
```

要求：

记录：

Total Frame Allocations

验证：
