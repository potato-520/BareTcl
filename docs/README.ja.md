# BareTcl 🚀

**世界最小クラスの産業グレードTclインタプリタ・コア**

BareTclは、リソースが極めて制限されたベアメタルMCU環境向けに設計された、超コンパクト、スタックレス、かつLibcフリーなTclインタプリタです。
